MODELLING DIFFUSION WITHIN DNA ORIGAMI NANOSTRUCTURES

To simulate the effect of the DNA origami on containing the intermediate species, and thus aiding the reaction kinetics, the diffusion of relevant species with the DNA origami was modelled using MATLAB PDE solver, utilizing finite element analysis.

This can solve PDEs of the form:

\[ m \frac{\partial^2 \varphi}{\partial t^2} + d \frac{\partial \varphi}{\partial t} - \nabla\cdot \left( \vec{c} \nabla\varphi \right) + a \varphi = f \]
[EQUATION 1]

which means that, for Fickâ€™s second law at steady state -

\[ 0 = D\nabla^2\varphi , \]
[EQUATION 2]

the parameters have to be set to \[ m = 0 , d = 0 , a = 0 , f = 0 \]

and

\[ \vec{c} = D \begin{bmatrix} D & 0 & 0 \\ 0 & D & 0 \\ 0 & 0 & D \end{bmatrix} \]

for isotropic diffusion with diffusion coefficient, D.

The simulation was able to run and provide qualitative results, however to provide accurate and properly scaled results using a fine enough mesh, more computational power would be required than was available and hence was not pursued further.
